## 摘要
尽管大语言模型（LLMs）推动了 CT 报告生成的发展，但现有方法通常对 3D 体数据进行整体编码，未能将关键信息线索与冗余的解剖背景区分开来。受放射学认知减影（radiological cognitive subtraction）的启发，我们提出了差异化视觉提示（DiffVP），它将报告生成建立在显式的、高层级的扫描与参考图像间的语义差异之上，而非仅仅依赖于绝对视觉特征。DiffVP 采用分层差异提取器，将互补的全局和局部语义差异捕捉到共享潜空间中，并配合差异到提示生成器，将这些信号转换为可学习的视觉前缀标记，用于 LLM 的条件引导。这些差异提示作为结构化的条件信号，隐式地抑制了不变的解剖结构，同时放大了具有诊断相关性的视觉证据，从而在无需显式病灶定位的情况下促进了准确的报告生成。在两个大规模基准测试中，DiffVP 始终优于先前的方法，平均 BLEU-1-4 分别提高了 +10.98 和 +4.36，并进一步提升了在 RadGenome-ChestCT 上的临床效能（F1 分数为 0.421）。所有代码将发布在 https://github.com/ArielTYH/DiffVP/。

## Abstract
While large language models (LLMs) have advanced CT report generation, existing methods typically encode 3D volumes holistically, failing to distinguish informative cues from redundant anatomical background. Inspired by radiological cognitive subtraction, we propose Differential Visual Prompting (DiffVP), which conditions report generation on explicit, high-level semantic scan-to-reference differences rather than solely on absolute visual features. DiffVP employs a hierarchical difference extractor to capture complementary global and local semantic discrepancies into a shared latent space, along with a difference-to-prompt generator that transforms these signals into learnable visual prefix tokens for LLM conditioning. These difference prompts serve as structured conditioning signals that implicitly suppress invariant anatomy while amplifying diagnostically relevant visual evidence, thereby facilitating accurate report generation without explicit lesion localization. On two large-scale benchmarks, DiffVP consistently outperforms prior methods, improving the average BLEU-1-4 by +10.98 and +4.36, respectively, and further boosts clinical efficacy on RadGenome-ChestCT (F1 score 0.421). All codes will be released at https://github.com/ArielTYH/DiffVP/.

#### 1. 核心问题与整体含义
在自动生成 CT 报告的任务中，现有的基于大语言模型（LLM）的方法通常将 3D CT 图像作为一个整体进行编码。然而，3D CT 数据具有**稀疏性**（病变区域仅占极小部分）和**冗余性**（充满大量正常的解剖背景）。LLM 往往难以从海量背景噪声中识别出关键的病理线索。
本文受放射科医生“认知减法”诊断逻辑的启发——即通过对比当前扫描与“正常参考”来识别异常——提出了 **DiffVP** 框架。其核心含义是：不再让模型孤立地看一张图，而是通过显式建模“输入图”与“正常参考图”之间的**语义差异**，引导 LLM 关注临床相关的细微变化。

#### 2. 方法论
DiffVP 包含两个核心模块，旨在将视觉差异转化为 LLM 可理解的提示（Prompt）：
*   **分层差异提取器 (HDE)：**
    *   **全局增量算子 (Global Delta Operator)：** 使用共享的查询（Query）通过 Transformer 架构捕捉输入图与参考图之间的整体结构和语义偏移。
    *   **局部增量算子 (Local Delta Operator)：** 基于欧几里得距离计算对齐后的视觉 Token 之间的残差，并进行几何加权聚合，从而捕捉局部细微的病变特征。
*   **差异到提示生成器 (DPG)：**
    *   将提取到的全局和局部差异信号进行融合，并通过投影层转换为一组可学习的**视觉前缀 Token (Visual Prefix Tokens)**。
    *   这些前缀 Token 与原始视觉 Token 一起输入到经过 LoRA 微调的 LLaMA-2-7B 模型中，作为“软提示”引导报告生成。
*   **诊断语义引导：** 引入一个预训练的分类器预测 18 种疾病标签，作为额外的语义锚点增强诊断的一致性。

#### 7. 优点
*   **临床直觉驱动：** 模拟放射科医生的对比诊断逻辑，具有很强的可解释性。
*   **分层建模：** 同时兼顾全局结构变化和局部细微病变，设计精巧。
*   **鲁棒性强：** 即使参考池非常小（仅需 10 个正常样本）或存在一定的领域偏移，模型依然能保持较好的性能提升。

*   **结构敏感性：** ROUGE-L 指标提升不显著，反映出模型在生成报告的段落顺序和叙述结构上与真值仍有偏差。

*   **应用限制：** 需要预先筛选出一小部分“正常”扫描作为参考库，这在某些极罕见病或特定场景下可能需要额外的人工筛选成本。
